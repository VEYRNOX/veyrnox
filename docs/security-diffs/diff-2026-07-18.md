The Codex P1 fix is a genuine correctness repair (without it, KEK-enrolled vaults would fail the auth-tag on every unlock). Covered by `vault-aad.test.js` (14 tests) + KEK regression tests (40 tests per CLAUDE.md). No "verified" claim made without evidence.
